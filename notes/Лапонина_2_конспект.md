# Лапонина О. Р. - Защита информации - Алгоритмы симметричного шифрования. DES, Blowfish

### Алгоритмы симметричного шифрования
+ **P** - plain text
+ **E** - encryption function
+ **D** - decryption function
+ **K** - key
+ **C** - cypher text

```
  K ->                            |    K  ->  
          ->  E(P, K)  -> C       |            ->  D(C, K) -> P
  P ->                            |    C  ->
```
